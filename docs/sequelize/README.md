## 本书简介

本书一共八个章节，分别讲述了如何使用 TypeScript 正确编写 Sequelize 模型，如何正确的书写 TS 代码提示接口，如何定义模型以及字段类型，如何定义模型之前的关联关系，怎样理解模型之间的关系，什么是模型关系的多态，如何对查询条件进行限定，生命周期函数，数据库迁移等。
